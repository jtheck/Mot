About

The internet's favorite rest stop for wayward Arduino, ESP32, and ESP8266 development boards and projects!

Mot.moe is an open source robotics and AgI platform founded on the Arduino and web open hardware software ecosystem. Build and share!


Resources & Tips

  1. Install the Arduino IDE appropriate for your computing environment.
  2. Select your Board. Use Board Manager to install it if it is missing.
  3. Use Manage Libraries to install and update required Project Libraries.
  4. Plug in Board, and verify Port selection for upload.
  5. Compile and Upload Code. Observe, compose, revise.
  6. Jiggle wires, reset power, and apply mild blunt force trauma as necessary.
